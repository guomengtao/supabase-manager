Project Setup

Learn how to set up your Supabase project and configure it for use with Supabase Manager. This guide will walk you through the essential steps.

Step 1: Create a Supabase Project

First, you'll need to create a new project in Supabase:

  1. Go to the Supabase Dashboard
  2. Click "New Project"
  3. Fill in your project details
  4. Choose a database password
  5. Select your region
  6. Click "Create Project"

Step 2: Get Your API Credentials

Once your project is created, you'll need to get your API credentials:

  1. In your project dashboard, go to "Settings" â†’ "API"
  2. Copy your "Project URL"
  3. Copy your "anon" public API key
Important: Never share your service_role key. We'll only use the anon key for this setup.

Step 3: Configure Supabase Manager

Now, let's configure Supabase Manager with your credentials:

  1. Open your config.js file
  2. Update the supabaseUrl with your Project URL
  3. Update the supabaseKey with your anon key
// config.js
const CONFIG = {
    supabaseUrl: 'YOUR_PROJECT_URL',
    supabaseKey: 'YOUR_ANON_KEY'
};

Step 4: Initialize Database Tables

Supabase Manager needs certain tables to function. Our initialization script will create these automatically:

  • users - For user management
  • files - For file tracking
  • images - For image management
  • articles - For content management

The tables will be created automatically when you first run the application. You can verify their creation in your Supabase dashboard under "Table Editor".

Success! Your project is now set up and ready to use with Supabase Manager. Continue to the Quick Start guide to learn how to use the basic features.
